PERSONNALISATION RAPIDE

1. Photos
- Remplacer les fichiers dans assets/img/
- Garder les mêmes noms pour éviter toute autre modification
- Pour changer l'ordre ou le style de la galerie, modifier assets/js/data.js

2. Prix, DPE, téléphone, email, localisation
- Modifier directement assets/js/data.js

3. PDF
- Remplacer assets/docs/fiche-appartement-vitre.pdf
- Ou changer le chemin dans assets/js/data.js

4. Carte
- Modifier la valeur mapEmbedUrl dans assets/js/data.js

5. Formulaire
- Dans contact.html, remplacer l'URL Formspree par la vôtre :
  https://formspree.io/f/REMPLACEZ_ICI_VOTRE_ID

6. Textes longs
- Modifier index.html, infos-pratiques.html et contact.html
